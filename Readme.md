The big picture
The project has for a goal to create a suite of sensors that would be representative of a fictional scientific expedition's away box.
The goal is to have a varied range of sensors, in self-contained modules, that will fit inside of a suitcase which acts like a home base.
The sensor modules will be battery powered and be fully independent, collecting and storing data. When the sensors are docked inside the briefcase, they will recharge their internal batteries and transfer data back to a central storage system. The central storage system can then transfer the data to a server - likely located in the fictional away team's vehicle.

The smaller picture: the temperature probe module
This specific module has the following goals:
-Fully waterproof outer housing
-Wireless charging
-Battery allowing for a reasonable data collection time
-Way to wirelessly configue the data capture rate
-RTC to timestamp readings
-Data storage to cover one full battery charge worth of data collection
-Wireless data transfer
